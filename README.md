Name: Activo Fijo Management

PHP 7.3 base project for asset management – ​​no framework, just the base structure.

## 🛠️ Requisitos Previos

Antes de instalar, asegúrate de contar con los siguientes elementos:

* **PHP:** Versión 7.0 o superior.
* **Servidor Web:** Apache / Nginx o el servidor embebido de PHP.
* **Base de Datos:** MySQL / MariaDB.
* **Gestor de Dependencias: Nope
